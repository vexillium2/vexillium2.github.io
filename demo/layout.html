Layout

vexillium2026/7/7小于 1 分钟GuideLayout

The layout contains:

  • Navbar
  • Sidebar
  • Footer

Also each page can contain:

  • BreadCrumb
  • Title and information
  • TOC (Table of Contents)
  • Meta information including update time and contributors
  • Comments

The theme also has the following elements:

  • Dark mode button
  • Back to top button
  • Print button

You can customize them in theme options and page frontmatter.
